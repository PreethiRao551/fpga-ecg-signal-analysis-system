## 📄 Research Publication

### Design of FPGA-Based ECG Signal Analysis System

- Published in **JOIREM Journal (Volume 04, Issue 07)**
- Link: https://joirem.com/jr-paper_issue/?select_volume=04&select_issue=7

### 📌 Abstract
This work presents a real-time ECG signal analysis system implemented on FPGA using Verilog HDL. The system performs signal filtering, QRS detection, and heart rate calculation with high accuracy and low latency.

### 🎯 Key Contributions
- FPGA-based hardware implementation of ECG processing
- QRS complex detection algorithm
- Real-time heart rate computation
- Optimized for low-power biomedical applications
